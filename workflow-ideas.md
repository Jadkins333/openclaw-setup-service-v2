# First Workflow Ideas

## 1. Ship a simple landing page
- create app
- edit copy
- build
- publish
- verify URL

## 2. Fix a project blocker
- inspect repo
- identify failure
- patch
- build/test
- summarize cleanly

## 3. Get one admin task done
- identify desired outcome
- choose shell/browser path
- execute carefully
- document the result

## 4. Make the setup pay for itself
- create one sellable asset
- publish it
- write first outreach copy
- send traffic
