# Starter Config Notes

## Trusted Machine Baseline
Use stricter settings on shared or risky machines.
Use more permissive settings only on your own trusted machine.

## Good default mindset
- internal work can move fast
- external/public actions deserve more review
- write down what changes so future-you is not blind

## Important concept
The effective host exec policy may be controlled by both main OpenClaw config and the host approvals file.
If the approvals file is stricter, you still get prompts.
