# OpenClaw Quickstart Checklist

## Goal
Get from install to a usable setup with the fewest moving parts possible.

## Checklist
- [ ] OpenClaw is installed and the CLI responds
- [ ] Gateway starts and stays healthy
- [ ] Your agent responds in the intended chat surface
- [ ] One file read works
- [ ] One shell command works
- [ ] Approvals are configured in a way that matches your machine trust level
- [ ] Browser capability is verified if you need authenticated web work
- [ ] Memory file locations are understood
- [ ] One real workflow completes end-to-end

## Definition of done
A setup is not “done” because the install succeeded.
A setup is done when you can use it for one meaningful task without getting stuck immediately.
